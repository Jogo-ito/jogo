BACKLOG v1.6 — ITO
==================

ITENS EM DESENVOLVIMENTO
--------------------------

[1] Filtro Anti-XSS
    Prioridade: ALTA / Segurança crítica
    Problema: campos input-nome enviam dados brutos para o Supabase sem sanitização.
    Solução: função sanitize() que escapa HTML antes de salvar/renderizar.
    Status: PENDENTE

[2] Trava de "Penetras"
    Prioridade: ALTA / Lógica de sala
    Problema: usuário pode entrar via URL direto mesmo com partida em andamento (status !== 'esperando').
    Solução: checar status da sala antes de inserir novo jogador; exibir mensagem de erro e bloquear entrada.
    Status: PENDENTE

[3] Vazamento de Conexões Realtime (WebSocket)
    Prioridade: ALTA / Performance e estabilidade
    Problema: inscreverNoJogo() não remove canais antigos se chamada mais de uma vez → memory leak.
    Solução: guardar referência do canal ativo e chamar .unsubscribe() antes de criar novo.
    Status: PENDENTE
